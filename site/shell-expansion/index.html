Regex

  • "?" : remplace 1 lettre
  • "*" : 0 ou plusieurs lettres
  • [abc]: un caractère parmi a, b ou c
  • [!abc]: un caractère saus a, b ou c
  • [[:alpha:]]: une lettre
  • [[:lower:]]: une lettre minuscule
  • [[:upper:]]: une lettre majuscule
  • [[:digit:]]: un chiffre
  • ^: début de ligne
  • $: fin de ligne
  • touch hugo{0..9}: crée un fichier hugo0, hugo1, ..., hugo9
